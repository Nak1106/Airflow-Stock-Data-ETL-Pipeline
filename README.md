# Airflow Stock Data Pipeline

## Project Overview
This project implements an **ETL pipeline** using **Apache Airflow** to fetch stock data from the **Alpha Vantage API**, transform it, and load it into **Snowflake**.

### Technologies Used:
- Apache Airflow
- Snowflake
- Alpha Vantage API
- Python
- SQL Transactions
